# 设备与分类管理手册

本手册详细介绍了模拟器中的设备分类管理和单个 IoT 设备管理功能。

## 1. 分类管理 (Category Management)
**分类管理器** 用于定义设备的模板。分类定义了一类设备（如“柴油发电机”、“温度传感器”）的通用指标（参数）、物理属性和默认视觉效果。

### 1.1 进入分类管理
- 在主侧边栏中点击 **"分类管理"** (Category Management) 标签。

### 1.2 创建分类
1. 点击 **"新建分类"** (Create Category)。
2. **基本信息**: 输入名称 (Name) 和代码 (Code，唯一标识符)。
3. **参数定义**: 定义设备上报的数据指标。
   - **Tag (标签)**: 静态元数据（如序列号、位置）。
   - **Metric (指标)**: 时间序列数据（如温度、转速）。
   - 设置数据类型 (整数、浮点数、字符串、布尔值) 和单位。
4. **物理配置**: 设置物理常量（如质量、最大转速）。
5. **视觉配置**: 选择默认的 3D 模型类型。

### 1.3 AI 生成
- 点击 **"AI 生成"** (AI Generate) 按钮，输入描述（例如“一个六轴工业机械臂”），大模型将自动为您生成完整的分类结构。

## 2. 设备管理 (Device Management)
**设备管理器** 是控制模拟设备的中心枢纽。

### 2.1 设备列表
- 查看所有设备及其状态（运行中/已停止）和当前场景。
- **筛选**: 使用搜索栏或状态过滤器查找特定设备。
- **批量操作**: 勾选多个设备进行批量启动、停止或删除。

### 2.2 添加设备
1. 点击 **"添加设备"** (Add Device)。
2. **手动模式**:
   - 选择一个分类。
   - 填写设备详情（ID、名称）。
   - 如有需要，覆盖默认参数。
3. **AI 批量生成**:
   - 描述您需要的设备（例如“仓库用的10个温度传感器”）。
   - 系统将自动批量生成这些设备。

### 2.3 视觉模型配置
- 在设备列表中，点击 **"视觉"** (立方体图标)。
- 配置 3D 展示效果：
   - **类型**: 立方体、圆柱体、球体或自定义 GLB/GLTF 模型。
   - **颜色/尺寸**: 自定义外观。
   - **映射**: 将视觉属性（如旋转角度、颜色变化）绑定到设备指标，实现实时可视化。
