# 仿真与场景手册

本手册说明如何运行仿真、管理场景以及与 3D 数字孪生环境进行交互。

## 1. 数字孪生可视化 (Digital Twin Visualization)
**数字孪生** 视图为您提供了 IoT 环境的实时 3D 展示。

### 1.1 导航控制
- **旋转**: 左键点击并拖动。
- **平移**: 右键点击并拖动。
- **缩放**: 鼠标滚轮滚动。
- **选择**: 点击 3D 场景中的设备，可查看其实时遥测数据浮窗。

### 1.2 设备指示
- 设备根据状态显示不同颜色：
   - **绿色**: 正常运行。
   - **红色**: 错误/临界状态。
   - **灰色**: 已停止/离线。

## 2. 场景管理 (Scenario Management)
场景定义了设备随时间变化的行为。场景允许您模拟特定条件（如“过热”、“振动尖峰”或“电池耗尽”），而非仅仅生成随机数据。

### 2.1 应用场景
1. 进入 **设备管理**。
2. 选择一个设备。
3. 在 **"场景" (Scenario)** 下拉菜单中，选择一个预定义的场景。
4. 设备输出的数据将立即反映该场景定义的逻辑。

### 2.2 创建场景
1. 打开 **场景管理器**。
2. 点击 **"新建场景"**。
3. **定义逻辑**:
   - 设置目标指标。
   - 定义函数（线性增长、正弦波、随机游走、阶跃变化）。
   - 设置持续时间和循环设置。
4. **AI 生成**: 您可以要求 AI “创建一个模拟冷却液泄漏的场景”，系统将自动生成相应的参数曲线。

## 3. 回放模式 (Playback Mode)
模拟器支持回放历史数据。

1. 切换顶部栏的 **"回放" (Playback)** 开关。
2. 选择一个 **时间范围**。
3. 使用 **时间轴滑块** 拖动进度。
4. 3D 场景和图表将更新，以反映系统在该特定时间戳的状态。
